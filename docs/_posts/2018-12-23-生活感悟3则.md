# 消费

# 真相
18年是明星不断跌落神坛人设崩塌的一年, 说明大众相比以前更加接近真相. 
但百度离真相越来越远, 搜索敏感词汇都不在有根据相关法律法规的提示了, 而是只显示某些预设的结果. 快消品当道,各种搜狐,头条等等垃圾号发布的垃圾文章阅读量上十万百万, 占在百度搜索的前几页, 而真正有心的有趣的有内容的文章, 被埋没在垃圾之中. 所以搜索,某些内容 不能局限在百度, 比如谷歌, 比如某些文献数据库, 都比百度有价值的多.
