# Nginx配置
代码如下:

```
yum install yum-utils
vim /etc/yum.repos.d/nginx.repo

[nginx-stable]
name=nginx stable repo
baseurl=http://nginx.org/packages/centos/$releasever/$basearch/
gpgcheck=1
enabled=1
gpgkey=https://nginx.org/keys/nginx_signing.key

[nginx-mainline]
name=nginx mainline repo
baseurl=http://nginx.org/packages/mainline/centos/$releasever/$basearch/
gpgcheck=1
enabled=0
gpgkey=https://nginx.org/keys/nginx_signing.key

yum install nginx
安装完后，rpm -qa | grep nginx 查看

启动nginx：systemctl start nginx
加入开机启动：systemctl enable nginx
查看nginx的状态：systemctl status nginx

配置文件:
vim /etc/nginx/conf.d/default.conf


修改nginx
vi /etc/nginx/conf.d/default.conf

server {
    listen  80;
    server_name exam.cyz.ink;
    location / {
        proxy_pass  http://127.0.0.1:8081;
    }
}

server {
        listen  80;
        server_name  blog.cyz.ink;
        access_log  logs/host.access.log  main;
        location / {
            root   /root/blog/public/;
            index  index.html index.htm ;
        }
        error_page   500 502 503 504  /50x.html;
        location = /50x.html {
            root   html;
        }
}

# 配置ssl

server {
        listen  80;
        server_name blog.cyz.ink ;
        access_log  logs/blog.access.log  main;
        rewrite ^/(.*) https://blog.cyz.ink permanent;
}

```

重启Nginx

```
/bin/systemctl restart nginx.service
```

# 配置Linux发送邮件

1. 关闭sendmail服务

```
#执行下面的命令，各位大侠都对号入座吧
#sendmial
service sendmail stop
chkconfig sendmail off
#postfix
service postfix stop
chkconfig postfix off

#再狠一点就直接卸载吧..
yum remove sendmail
yum remove postfix
```
2. 修改/etc/mail.rc，在文件末尾增加以下内容，指定外部的smtp服务器地址、帐号密码等
```
$ vi /etc/mail.rc  
set from=123456@qq.com
set smtp=smtp.qq.com  
set smtp-auth-user=123456
set smtp-auth-password=runoob
set smtp-auth=login  
```
3. 发送邮件

```
echo  hello word | mail -s " title" 123456@qq.com  
```
