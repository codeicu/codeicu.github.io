小龙虾看似是道大菜, 但做起来挺简单的. 
1. 虾洗净去虾线, 背部剪开以便入味
2. 小龙虾先过宽油, 让肉质紧致, 捞起留底油, 
3. 葱姜给足, 生抽老抽糖盐耗油胡椒粉, 麻辣味的话就下辣椒花椒大料或者火锅料, 蒜蓉味的话烧制时间过半时放入蒜蓉.
4. 啤酒给半瓶, 水不要放多, 不然就成水煮小龙虾, 味道不够了.
5. 小火多炒一会, 更加入味, Done

btw, 虾头中的虾黄是精髓, 掰开头壳嘬一口美滋滋!
