# 类的属性表示方式
类名、属性、方法

1. +:public
2. -:private
3. \#:protected
# 类方法的表示方式
可见性、名称（参数列表）、[:返回类型]

# 类与类之间的表示方式

> 关联关系
```
1. 单向关联（一方持有对方成员变量） 实线箭头
2. 双向关联（双方持有对方成员变量） 无箭头实线
3. 自关联 指向自身的实线箭头
```
> 聚合关系（部分可以脱离整体） 空心菱形+直线箭头
> 组合关系（部分不能脱离整体：头部和嘴的关系） 实心菱形+直线箭头
> 依赖关系（方法中含有其他类） 虚线箭头
> 继承关系（继承类） 空心三角+实线
> 接口对应关系（接口实现） 空心三角+虚线
