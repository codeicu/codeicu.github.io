1. DOM 是Document Object Model( 文档对象模型 )的缩写。 
2. DOM把所有的Html都转化为节点
```
DOM把所有的html都转换为节点
整个文档 是一个节点
元素 是节点
元素属性 是节点
元素内容 是节点
注释 也是节点 

例如
<html>
<body>
<div id="d1">hello HTML DOM</div>
</body>

var  div1 = document.getElementById("d1");
p("文档节点"+document);
p("元素"+div1);
p("属性节点"+div1.attributes[0]);
p("内容节点"+div1.childNodes[0]);

文档节点[object HTMLDocument]
元素[object HTMLDivElement]
属性节点[object Attr]
内容节点[object Text]
```
