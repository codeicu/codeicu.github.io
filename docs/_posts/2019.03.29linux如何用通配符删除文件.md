本文记录两点重要内容
1. 如何查看当前目录的大小
```
du -l
 -m 以Mb为单位
 -h 以K,M,G为单位  h是human的意思
 -s 仅显示总计
```
2. 如何用通配符删除文件
```
rm -f find 1-*
```
3. 用正则删除文件
```
find /opt/apache-tomcat-8.5.24/ -regextype sed -regex ".*ges.*-.*\.JPG" | xargs rm -f

效果：在tomcat文件夹下查找所有*ges*-*.JPG文件并将其删除
```
